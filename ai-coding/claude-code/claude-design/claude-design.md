## Claude Design
- Claude Designとは？
  - Anthropic Labsが2026年4月17日に発表した、自然言語による対話だけで高品質なビジュアル成果物（デザイン、インタラクティブなプロトタイプ、スライド資料など）を生成できるAIツール
  - 誰が使える？
    - ClaudeのPro、Max、Team、Enterpriseプランの加入者
  - どこから使える？
    - ブラウザ上の専用環境（claude.ai/design）にアクセスすることで利用可能
- Claude Design の強み(Figmaなどのデザインツールとの違いは？)
  - 学習コスト0、デザイン知識0でもハイレベルなデザインができる
    - チャットから指示するだけ
    - 細かなデザイン設定はUI上で選択できる
      - 手書き - ワイヤーフレーム - 完成版 のような{言葉がわからないので適切な言葉を教えて}も選択可能
  - Claude Codeとの互換性
    - ハンドオフ機能？
    - import機能
    - → 上を組み合わせて擬似的に双方向同期が可能
