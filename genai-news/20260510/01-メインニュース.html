GPT-5.5 Instant が既定化、ファミリー完成

2026/05/05

OpenAIはChatGPTのデフォルトモデルをGPT-5.5 Instantへ更新。法律・医療・金融などの高ステークス領域でハルシネーションを 52.5%削減 し、過去会話・ファイル・Gmailを参照した個人化応答が可能になった。AIME 2025は 81.2（旧65.4）、MMMU-Proは 76（旧69.2）。

「GPT-5.5 Instantは、法律・医療・金融などのセンシティブな領域においてハルシネーションを削減しながら、これまでと同等の低レイテンシを維持する」

Realtime API GA、音声3モデル登場

2026/05/07

OpenAIは音声推論の「GPT-Realtime-2」、70+→13言語の同時通訳「GPT-Realtime-Translate」、ストリーミング文字起こし「GPT-Realtime-Whisper」を一斉投入。Realtime APIはbetaを終え GA へ。GPT-Realtime-2は128Kコンテキストに拡張、Big Bench Audioで 96.6%（旧81.4%）を達成。

「GPT-Realtime-2はGPT-5クラスの推論を備えた最初の音声モデルであり、ツール呼び出しや割り込みを処理しながら自然に会話を継続できる」

EU、AI法簡素化で暫定合意

2026/05/07〜08

欧州理事会と欧州議会はAI法の簡素化案で暫定合意。高リスクAIシステムの規則適用を 2027年12月2日（バイオメトリクス・重要インフラ等）と 2028年8月2日（製品組込み）へ整理し、AI生成コンテンツ表示義務の猶予を 6→3カ月 に短縮。5月8日には透明性義務ガイドライン草案のコンサルテーションも開始。

「2026年8月2日以降、EU域内ではAIシステムとの相互作用やAI生成・操作コンテンツへの露出について利用者に通知することが義務化される」
Council and Parliament agree to simplify AI rules — Consilium [政府発表]
EU agrees to simplify AI rules — European Commission [政府発表]

米CAISI、Google・MS・xAIモデルを事前評価

2026/05/05

米商務省NIST傘下のCAISIが、Google DeepMind・Microsoft・xAIのフロンティアAIモデルを公開前に政府が評価する合意を発表。安全装置を解除した状態の能力評価を含み、機密環境でテスト する。OpenAI・Anthropicに続き、米国版の事実上の事前審査体制が拡大。

「独立した厳密な計測科学は、フロンティアAIとその国家安全保障上の含意を理解するために不可欠である」（CAISI所長 Chris Fall）

2026/05/06

AnthropicはSpaceXの「Colossus 1」（300MW超／22万GPU超）を独占活用する計算契約を締結。Claude Codeの5時間レート上限を有料プランで 倍増、Pro/Maxのピーク時間制限を撤廃、Claude Opus APIのレート上限も大幅引き上げ。Amazon・Google/Broadcom・MS+NVIDIAに続く大型インフラ提携。

Gemini API File Search マルチモーダル化

2026/05/04〜05

GoogleはGemini API File Searchをマルチモーダル化。Gemini Embedding 2 による画像とテキストの統合検索、カスタムメタデータ、ページ単位の引用に対応。さらにバッチ・動画生成・非同期タスクの完了を署名付きJSONで通知する イベント駆動Webhooks を追加し、ロングランニングジョブのポーリング不要を実現。
